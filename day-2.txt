Document Object: 
    A document object represents an HTML or XML document that is loaded in the browser. It provides an interface that allows JavaScript code to access
and manipulate the content of the document. The document object is a property of the window object, which means that it can be accessed using the 
window.document property.

Window Object: 
    A window object represents the current browser window or tab. It provides an interface that allows JavaScript code to access and manipulate 
    the properties and methods of the browser window.
